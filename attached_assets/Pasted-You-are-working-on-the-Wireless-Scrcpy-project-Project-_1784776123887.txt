Project purpose
---------------
Wireless Scrcpy is a Windows desktop application built with .NET 8 WinForms.

Its purpose is to provide a clean graphical interface for the entire wireless scrcpy workflow, including:

- Detect ADB
- Detect scrcpy
- Detect connected Android devices
- Enable TCP/IP mode
- Discover the phone's IP address
- Connect over Wi-Fi
- Launch scrcpy
- Disconnect and reconnect
- Display workflow status
- Run primarily from the Windows system tray

The application uses an ApplicationContext-based architecture instead of a traditional Program.Run(new Form()) design.
